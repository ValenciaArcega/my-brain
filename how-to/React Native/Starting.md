# React Native

Install the Explo CLI
```
npm install -g expo-cli
```

Create the app with Expo ✅
```
npx create-expo-app
```

Update the project
```
npm install -g create-expo-app
```

or we can use the CLI ❌
```
// 👉 start a new project
expo init project-name

// 👉 updates
npx expo upgrade
npx expo-cli upgrade  
npx expo install react-native@0.71.6
```

🚀
```
npx expo start
```
