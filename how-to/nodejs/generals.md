# Node

```bash
node -v || node --version
```

### NPM Updates

```
npm install -g npm@latest
```

## Update Node with NVM

Show all current node versions

```bash
nvm list available
```

Install an specific node version

```
nvm install 6.9.2
```

Switch version of node

```
nvm use 6.9.1
```

---

Delete cache of libraries

```
npm clean cache --force
```
